# Gmail UI Redesign
This app is created with command `npx create-react-app`.

## Deploy Locally
```bash
# Install dependencies
cd gmail-redesign-code
npm install @material-ui/core
yarn add uuid
yarn add react-router-dom
yarn add firebase
# Start locally
yarn start
```

## Deploy on Firebase
```bash
cd gmail-redesign-code
npm install -g firebase-tools

```
